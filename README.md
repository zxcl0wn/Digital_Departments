# Описание проекта
Проект представляет собой простую end-to-end систему, которая генерирует, сохраняет и визуализирует данные. Система состоит из генератора данных на Python, базы данных PostgreSQL и платформы Redash для визуализации и анализа.

Выбранная предметная область: **Игровая статистика**.

**Основные компоненты:**
1. Генератор данных
2. База данных PostgreSQL
3. Redash

---

# Инструкция по запуску

1. Клоинировать репозиторий
```
git clone https://github.com/zxcl0wn/Digital_Departments.git
```
2. Инициализировать базу Redash
```
docker compose run --rm redash create_db
```
3. Сбилдить и запустить контейнеры
```
docker compose up --build
```
